## 💡 How to Use This Repo

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yukti-says/logic_building
   cd logic-building
   ```

2. **Try a problem yourself first** — seriously, don't peek at solutions right away.

3. **Run a solution:**
   ```bash
   # Java
   javac Phase1_Conditionals/Level1/Q01.java && java Q01

   # Python
   python Phase1_Conditionals/Level1/Q01.py

   # C++
   g++ Phase1_Conditionals/Level1/Q01.cpp -o Q01 && ./Q01
   ```

4. **Track your progress** by checking off problems in the table above.
